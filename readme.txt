each chapter contains all files and migration files required to run the chapter.
before running code you need to run

1. $ composer update ( this could change based on chapter so better read chapters readme.txt file ) 

from the chapters root directory and also 

2. change database information 

and

3. $ php artisan migrate 

for getting all database tables.
